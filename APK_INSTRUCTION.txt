**New Premium APK use karne ka proper tareeka (IRCTC aur RailOne dono ke liye same process):**

Sabse pehle APK install karein. Agar **RailOne APK** hai to usko **SAI installer** ke through install karna compulsory hai.

APK open karte hi aapko **Tesla logo** dikhega. Isi logo ke through aapko saari settings manage karni hain aur token send karna hota hai.

Jab aap **Tesla logo par click** karenge, to wahan **3 options** show honge:

**1) Tokens**
Is option ke andar aapke saare generated tokens dikhai denge. Yahin par aapko apni **User ID enter** karni hoti hai aur **token send** karna hota hai.

**2) Device ID**
Is section me jaakar sabse pehle **Generate All** par click karein.
Token generate karne se pehle hamesha *Generate All* karna zaroori hai, taaki APK aapki **device details automatically change** kar sake.
Generate All karne ke baad APK ko **background se close** karein, phir **2–3 seconds baad dobara open** karein aur token generation start karein.

?? **Important notice:**
Har baar token banane se pehle **device details change** karna mandatory hai. Isse *unable to process* ya kisi bhi type ki error nahi aayegi aur token smoothly generate hoga.
Device details change karne ke baad **APK restart zaroor karein**, taaki app nayi settings ke saath properly initialize ho sake.

Baaki **token generation ka process bilkul same** rahega—payment page tak jaakar wahan se token send kar dena hai.

**Jade form booking** ya **Jade pair login** ke liye hamesha device details aur network change karke hi token generate karein.
